FileOpenerformClipboard
=======================

## 使い方

1. リンクをコピーする
2. アプリケーションを起動する

### 開けるリンクの例

URLとか

    http://google.com
    https://google.com

サーバー上のfolderとか

    \\server\folder

複数行で記述されたファイルとか

    \\server\folder
    filename.cpp

複数行で記述されたファイルとか2

    \\server\folder\
    filename.cpp

複数行で記述されたファイルとか3

    \\server\folder\
    subfolder\subfolder2
    filename.cpp
